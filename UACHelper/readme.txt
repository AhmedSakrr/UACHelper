						                
                             UACHelper

		A  helper  class  library  to detect,  manage  and use  UAC 
    functionalities in your program.

===================================================================

CLASSES

	UACHelper.UACHelper:   This static class contains properties 
    about  the current  state  of the application  as well  as 
    providing methods to get information about other processes 
    and starting new ones.

  UACHelper.AAMSettings:   This static class contains settings 
    of the "Admin Approval Mode"

  UACHelper.WinForm:   This  static  class  contains  multiple 
    methods  to add the UAC shield  icon to the buttons, forms
    and native dialogs
